# Artem Simbirkin - Product Delivery Portfolio

Public static site for GitHub Pages.

## Publish from GitHub

1. Create a public repository named `artem-simbirkin-portfolio`.
2. Upload the contents of this folder to the repository root.
3. Open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main` and folder `/(root)`, then save.
6. GitHub will publish the site at `https://<github-username>.github.io/artem-simbirkin-portfolio/`.

## Public scope

This package includes only the portfolio landing page, resume, case HTML files, and styles. Do not add internal source materials, evidence registries, client files, or environment files to this public repository.
